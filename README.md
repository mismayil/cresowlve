# CresOWLve: Benchmarking Creative Problem-Solving Over Real-World Knowledge

This is a bilingual benchmark for creative problem-solving grounded in real-world knowledge and solvable by human experts. 
CresOWLve spans a diverse range of knowledge and creative domains, varies in difficulty, requires multiple creative thinking strategies, and is manually validated to ensure quality. It contains ~2K open-ended questions with answers and explanations.
